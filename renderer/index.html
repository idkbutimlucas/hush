Où est Discord ?

Sur cette machine, ou sur un autre PC (setup double PC) — Hush le coupe dans les deux cas.

💡 Setup double PC ? Tu dictes sur ce Mac mais Discord tourne sur ton autre PC ? Installe Hush sur les deux : le PC qui a Discord devient l'hôte, celui où tu dictes devient le contrôleur. Les deux doivent être sur le même réseau (Wi-Fi/box).
Emplacement de Discord

Ce Mac va commander le Discord de l'autre PC. Trois étapes :

  1. Sur le PC qui a Discord : ouvre Hush, coche « Cette machine héberge Discord » → il affiche son IP et un code d'appairage.
  2. Ici, choisis l'hôte trouvé (ou saisis son IP) et recopie le code.
  3. Connecter. Ensuite, tiens ton raccourci : le Discord de l'autre PC se coupe.
